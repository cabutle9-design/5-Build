# 5-Build
Module 5 Build
The Indie Film Collective is a multi-page website meant to tell the community about its film screenings, workshops, collaborations, and volunteer opportunities.

The HTML is semantic and validated.

The CSS is mobile-first and responsive for larger screens.

There is no JavaScript.

# Pages Included

- index.html
- screenings.html
- workshops.html
- collaborations.html
- apply.html
